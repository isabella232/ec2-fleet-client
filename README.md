# ec2-fleet-client

client for ec2-fleet

## Getting Started
Install the module with: `npm install ec2-fleet-client`

```javascript
var ec2_fleet_client = require('ec2-fleet-client');
ec2_fleet_client.awesome(); // "awesome"
```
